# reference/ — справочные данные из синтакс-помощника 1С

## `public/reference/syntax-help.json`

Структурные **факты** о глобальных функциях и методах коллекций BSL, извлечённые
из выгрузки синтакс-помощника 1С: имена ru/en, сигнатуры, параметры
(тип/обязательность), возвращаемый тип, доступность. Лежит в `public/`, чтобы
панель-справочник грузила его лениво (`fetch`), не раздувая основной бандл.

Используется как:
- источник для будущей **панели-справочника** (синтакс-помощник §7);
- ориентир при реализации функций в рантайме (сверка сигнатур).

### ⚠️ Лицензия

Тексты синтакс-помощника — собственность «1С». Поэтому здесь **только факты**
(API-поверхность: имена, сигнатуры, типы), которые авторским правом не
защищены. **Описания и примеры 1С НЕ извлекаются** — их пишем свои.

Исходная выгрузка (~110 МБ) в репозиторий **не коммитится**.

### Регенерация

```bash
BSL_SP_EXPORT="/путь/к/Файл выгрузки.xml" npm run extract:sp
```

Скрипт: [`scripts/extract-syntax-help.ts`](../scripts/extract-syntax-help.ts).
Парсит справочник `ЭлементыДокументации` формата `_1CV8DtUD`, оставляет
глобальный контекст и типы коллекций, складывает факты в `syntax-help.json`.
